# watcher

```
NAME:
   watcher - watch file modified and execute something

USAGE:
   watcher {target-file} "{execute command}"

VERSION:
   0.0.1

GLOBAL OPTIONS:
   --help, -h     show help (default: false)
```


# install

## from source

```
git clone {this-repo}
cd {this-repo}
go install
```
